# QA: 오늘의 말 · 확장 선택 (v1.3.3)

## 목적
홈의 데일리가 **순서 클리어 7단계가 아니라** 구어 확장 선택판인지 검증한다.
연습 후에는 강제 다음 스텝이 아니라 같은 선택판으로 돌아와 **이어가기**를 유도하는지 확인한다.

## 검증 환경
- 브라우저: Chrome/Safari 최신 버전
- 해상도: 데스크톱 1280px+ / 모바일 390px
- 서버: `python3 -m http.server 8080`
- 캐시: `fresh.html`로 진입 권장

## E2E 테스트 케이스

### 1. 홈에 확장 선택판이 보인다
**시나리오**: localStorage 초기화 후 홈 로드

**예상 결과**:
- CTA `오늘의 말로 시작`
- 카피: 순서 클리어가 아니라 선택·확장
- 선택: 듣기 / 따라하기 / 이어묻기 / 이어답하기 / 이어말하기 / 화제전환
- 진행 표시는 `이어가기 N` (완료 N/7 아님)

**통과 조건**: ✅ / ❌

---

### 2. 시작 버튼은 부드러운 첫 확장(듣기)으로 들어가고, 이후는 선택
**시나리오**: `오늘의 말로 시작` 클릭 → 연습 마무리

**예상 결과**:
- 기본 진입은 듣기(강제가 아니라 입구)
- 마무리 후 **확장 선택판**으로 복귀 (「7/7 완료」 화면 없음)
- 다른 확장을 골라 같은 말·다른 말로 계속 가능

**통과 조건**: ✅ / ❌

---

### 3. 이어묻기 / 이어답하기
**시나리오**: 매트릭스가 있는 표현에서 이어묻기·이어답하기 선택

**예상 결과**:
- 이어묻기 → matrix question
- 이어답하기 → matrix shortYes (없으면 이어말하기 폴백)
- 성공 시 `quest:expand:…` 토큰·`expandCount` 증가

**통과 조건**: ✅ / ❌

---

### 4. 화제전환
**시나리오**: 화제전환 선택

**예상 결과**:
- 가까운(또는 다른) 표현으로 바뀌어 말하기 연습
- `lastExpressionId`가 새 표현으로 갱신
- 이후 선택판도 새 표현 기준

**통과 조건**: ✅ / ❌

---

### 5. 이어가기는 끝나지 않는다
**시나리오**: 확장을 7회 이상 반복

**예상 결과**:
- 링/퍼센트는 100% 근처까지 채워질 수 있음
- 그래도 선택판이 남아 「더 이어가기」 가능
- `isCompleted`로 잠기지 않음

**통과 조건**: ✅ / ❌

---

### 6. 날짜 변경
**시나리오**: `dailyQuestV1.date`를 어제로 두고 새로고침

**예상 결과**:
- 새 seed·`expandCount` 0으로 리셋
- 선택판이 다시 보임

**통과 조건**: ✅ / ❌

## 자동화 스모크 (선택)
