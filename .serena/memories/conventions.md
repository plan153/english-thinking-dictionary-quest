# Conventions
- Keep app vanilla and dependency-free.
- Prefer Korean learner-facing copy that is easy and natural.
- Data model emphasizes `coreVerbId` + `patternId` + `nounIds` + `situationTags`; expressions should not be standalone vocabulary entries.
- Expression cards should include 3 hints and support quiz derivation through `quizTypes`, `chunks`, and `audioText`.
- Preserve existing inline app style unless a larger refactor is explicitly requested.
