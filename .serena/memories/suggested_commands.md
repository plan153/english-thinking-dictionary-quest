# Suggested Commands
- Run validation: `python3 scripts/validate.py`.
- Serve locally for JSON fetch/PWA behavior: `python3 -m http.server 8081` if 8080 is busy.
- Check JSON syntax: `python3 -m json.tool data/expressions.json` and same for other data files.
- Inspect files quickly: `rg --files`, `rg <pattern>`.
